Decide the type of person you want to be

The more efficient way to stick to your habits on the long run and reach your goals is acutally not :

1. Set your goal 2. Define habits accordingly 3. Change thanks to these habits 4. Reach your goals

But rather :

1. Decide the type of person you want to be a. You believe b. You think c. You behave

Differently because you are this person by now.

 2. Define habits your habits accordingly 3. Prove yourself you're this person everytime you implement a habit 4. Observe long term results

So let's define the type of person you have decided to be.

Basically a type of person or an Identity is a set of Categories to which you can assign Importance.

Let's say, you want to run a marathon, that's to say want want to be a marathonian. You can define an Indentity called "Marathonian" and assign the Category "Athletic" with an Importance of 100% and the Category "Relax" with an Importance of 30% because let's say that after running you feel more relaxed.

The idea is to associate a Habit with a list of Identities.

Everytime you achieve this Habit it's a vote for these Identities you want to become.
